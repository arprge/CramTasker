# Project Context: Personal Exam Prep Tasker (C++)
**Course Duration:** 15 weeks (University Undergraduate Level)
**Objective:** Build a Task Scheduler that optimizes study time based on exam deadlines, subject difficulty, and credits.

## Academic Requirements:
- **Data Structures (Target: 4-5):** Must eventually include Vector, Hash-Table (unordered_map), Priority Queue (Heap), and potentially Trees or Graphs.
- **Algorithms (Target: 3-4):** Must include Sorting (O(n log n)), Greedy (Activity Selection), and Dynamic Programming (Weighted Interval Scheduling).
- **Architecture:** Professional multi-file structure (.h / .cpp), clean separation of concerns, and Big-O complexity awareness.

## Key Features for the Roadmap:
1. Integration with Google Calendar via iCal (.ics) link (Downloader + Parser).
2. Productivity heuristics (User-specific peak performance hours).
3. GPA Optimization: Prioritizing tasks where (Credits * (100 - CurrentGrade)) is highest.

## Current Constraint:
The project is at an early stage. We don't need a "ready-to-run" monolithic app. We need a robust, scalable foundation (Skeleton) that follows the university syllabus but looks professional.
